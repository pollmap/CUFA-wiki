# 주식 분석 처음이에요 — 14일 로드맵

<WhatYouWillLearn>
- 재무제표 3표(I/S, B/S, C/F)의 역할과 상호 연결을 읽어낼 수 있다
- PER / PBR / ROE를 직접 계산하고 상대 비교로 저평가를 가려낼 수 있다
- 내가 아는 기업 하나를 골라 DCF 민감도 테이블까지 포함한 **첫 리포트**를 작성한다
</WhatYouWillLearn>

**대상 독자.** 주식 투자를 "해보긴 했는데" 기업을 **숫자로 읽어본 적이 없는** 사람. 뉴스와 루머가 아닌 **재무제표와 공시**로 의사결정하는 습관을 만든다.

**매일 30-60분 × 14일 = 총 7-14시간.** 끝나면 [기업·산업 허브](/industries)에서 관심 종목 하나를 실제로 분석한다.

---

## 학습 경로 개요

14일은 **2주 구조**. Week 1은 "회계 언어 해독" (투자자 관점의 재무제표 읽기), Week 2는 "밸류에이션 적용" (상대가치 + DCF 입문).

- **전제 지식.** 없음. 한국 고등학교 수학 (수 I 수준) 으로 충분.
- **필요 도구.** 브라우저 + [DART 전자공시](https://dart.fss.or.kr) 계정 (무료).
- **선택 도구.** 엑셀 / Google Sheets — Week 2 Day 11에서 DCF 모델링 시 사용.

<KoreaContext>
**한국 시장 특수 고려사항.**

1. **KOSPI + KOSDAQ 이원 구조.** 대형주는 KOSPI(삼성전자·현대차·SK하이닉스), 성장·바이오는 KOSDAQ(카카오·셀트리온제약·에코프로비엠). 첫 기업으로는 **본인이 소비자로서 매일 쓰는 회사** 추천 — 예: 카카오(플랫폼), 현대차(자동차), CJ제일제당(식품).

2. **연결 vs 별도 재무제표.** 재벌 계열사를 볼 때는 **반드시 연결(CFS)** 사용. 별도(OFS)는 지주회사 분석 외엔 부차적. DART 조회 시 "연결재무제표" 라벨 확인.

3. **DART 공시 3종 세트.** 사업보고서(연 1회) · 반기/분기보고서(분기별) · 주요사항보고서(수시). 초보자는 사업보고서 **II. 사업의 내용 + III. 재무에 관한 사항** 두 섹션만 읽어도 충분.

4. **원·달러 환율 영향.** 수출기업(삼성전자·현대차)은 환율 상승 = 실적 호재, 수입의존 기업(항공·화학원재료)은 반대. 손익계산서의 **외환차이/환산이익**은 경상실적과 구분.

5. **주주친화성 격차.** 한국 상장사 평균 배당성향 약 20% (미국 40%+). 배당주 투자 시 "코리아 디스카운트" 감안, 자사주 매입·소각 공시는 DART "자기주식 취득" 검색.
</KoreaContext>
