# 퇴직연금 굴리기 — 5일 로드맵

<WhatYouWillLearn>
- DB · DC · IRP의 **세제·운용·리스크** 차이를 한 문장으로 설명할 수 있다
- TDF(Target Date Fund)의 글라이드 패스 개념과 2030 / 2040 / 2050 빈티지 선택 기준을 안다
- 본인 연령대에 맞는 **공격형·중립형·안정형** 3개 템플릿을 직접 구성한다
- 연금 수령 단계에서의 **세금·인출률·장수 리스크** 를 이해한다
</WhatYouWillLearn>

**대상 독자.** 직장인 / 사회초년생 / 퇴직예정자 모두. 퇴직연금을 **"그냥 원리금보장"** 으로 방치 중인 80% 를 위한 실전 가이드.

**매일 30-45분 × 5일 = 총 2.5-4시간.** 끝나면 본인 퇴직연금 계좌의 상품 구성을 **직접 바꿀 준비** 가 된다.

---

## 학습 경로 개요

5일은 **집중형 구조**. Day 1-2는 제도 이해(DB/DC/IRP + TDF), Day 3-5는 포트폴리오 구성·리스크 관리·인출 전략.

- **전제 지식.** 권장: [ETF로 시작](/learn/paths/etf-first) 로드맵 완주. 필수는 아니지만 자산배분 감각이 있어야 Day 3이 쉬워짐.
- **실습 준비.** 본인이 가입된 운용사(미래에셋·삼성·한투·KB 등) 퇴직연금 앱 로그인 가능 상태.

## Day-by-day

### Day 1 — DB vs DC vs IRP 차이 이해

- **학습.** [/learn/foundation/overview](/learn/foundation/overview) (재무제표 기초가 연금 자산 이해의 출발점)
- **핵심 구분.**
  - **DB(확정급여)** — 회사가 운용 리스크 부담, 근로자는 퇴직 전 임금×근속연수 공식으로 수령
  - **DC(확정기여)** — 근로자가 운용 리스크 부담, 매년 회사가 입금한 적립금을 **본인이 운용**
  - **IRP(개인형)** — 이직·퇴직 시 DC가 이동, 추가 납입 시 **연 최대 900만원(연금저축 합산) 세액공제**
- **실습.** 본인 계좌가 DB/DC/IRP 중 무엇인지 확인. DC/IRP 라면 **본 로드맵의 실제 실습 대상**.

### Day 2 — TDF · 글라이드 패스 · 빈티지 선택

- **학습.** [/learn/portfolio/portfolio-construction](/learn/portfolio/portfolio-construction)
- **TDF 원리.** 은퇴 목표연도(빈티지)에 맞춰 **주식 비중이 자동으로 감소** 하는 펀드. 2060 빈티지 = 청년용(주식 80%), 2030 빈티지 = 은퇴 임박(주식 30%).
- **글라이드 패스.** 시간 경과에 따라 주식 → 채권으로 **자동 리밸런싱 궤적**. 국내 TDF는 대개 은퇴 시점 주식 30-40%, 이후 10년간 20%까지 점진 감소.
- **실습.** 본인 은퇴 예상연도 = 만 60세 시점. 예: 현재 30세라면 **2055 TDF** 가 가이드 빈티지.

### Day 3 — 퇴직연금 포트폴리오 구성 (3개 템플릿)

- **학습.** [/learn/portfolio/correlation-diversification](/learn/portfolio/correlation-diversification) + [포지션 사이징](/learn/portfolio/position-sizing)
- **3개 템플릿 (참고).**
  - **안정형 (50대+)** — 채권ETF 40% + 원리금보장 30% + 주식ETF 20% + 대체 10%
  - **중립형 (30-40대)** — 글로벌주식ETF 50% + 채권ETF 30% + 국내주식ETF 15% + 대체 5%
  - **공격형 (20-30대)** — 미국S&P500 40% + 선진국ETF 20% + 신흥국ETF 10% + 국내주식 10% + 채권 15% + 리츠 5%
- **원리금보장 상품 유의.** 원리금보장 = 은행예금 형태, 수익률 연 2-3%대. **인플레이션 보전 불가** 를 명심하고 비중 신중히 설정.
- **실습.** 본인 템플릿 초안 확정 + 운용사 앱에서 실제 선택 가능한 상품 목록과 대조.

### Day 4 — 리스크 관리 · 드로우다운 대응

- **학습.** [/learn/portfolio/drawdown-management](/learn/portfolio/drawdown-management) + [스트레스 테스트](/learn/portfolio/stress-testing) + [위험조정수익률](/learn/portfolio/risk-adjusted-returns)
- **퇴직연금 특유의 리스크.**
  - **시퀀스 오브 리턴 리스크(Sequence Risk)** — 은퇴 직전 5년 하락장이면 타격 치명적
  - **장수 리스크(Longevity Risk)** — 100세 시대, 적립금이 먼저 고갈될 수 있음
  - **인플레이션 리스크** — 연금의 실질 구매력 감소
- **실습.** 본인 포트폴리오의 **은퇴 시점 -30% 시나리오** 후 회복 기간 가늠.

### Day 5 — 인출 전략 · 세금 최적화 · 점검 주기

- **학습.** [/learn/portfolio/stop-loss](/learn/portfolio/stop-loss) (철학적 참고) + [/learn/portfolio/behavioral-finance](/learn/portfolio/behavioral-finance)
- **인출 전략 3종.**
  - **4% Rule** — Bengen(1994). 첫해 적립금의 4% 인출, 이후 인플레 조정. 30년 안전기준.
  - **동적 인출(Guyton-Klinger)** — 시장 상황에 따라 인출률 조정
  - **Bucket 전략** — 단기(현금 2-3년치) + 중기(채권) + 장기(주식) 3개 주머니
- **세제 혜택 정리.**
  - 납입: 연 최대 900만원 세액공제 (연금저축+IRP 합산, 16.5% or 13.2%)
  - 운용: 과세이연
  - 인출: 55세+ 10년 이상 유지 시 **연금소득세 3.3-5.5%** (일반소득세 대비 유리)
- **점검 주기 권장.** 분기별 **자산배분 확인**, 연 1회 **리밸런싱**, 5년마다 **목표·템플릿 재검토**.
- **실습.** 리밸런싱 규칙이 명시된 한 장짜리 **연금 IPS** 작성.

<KoreaContext>
**한국 퇴직연금 특수 고려사항.**

1. **원리금보장 편중.** 2023년 기준 국내 퇴직연금의 약 85% 가 원리금보장 상품에 묶여있음. 연 2-3% 수익률은 인플레이션 보전도 안되는 수준이라 **실질 구매력 감소**. DC/IRP 가입자는 실적배당형 전환 적극 검토.

2. **디폴트옵션 제도 (2022.7 도입).** DC/IRP 가입자가 특정 상품을 고르지 않아도 **지정된 TDF/BF/MMF** 가 자동 운용됨. 각 운용사가 디폴트옵션 상품군을 사전 고지. 방치 중이라면 **디폴트옵션 지정 여부** 부터 확인.

3. **연금저축 + IRP 합산 900만원 한도.** 세액공제 총 한도 900만원 (연금저축 600만원 + IRP 300만원 까지 합산). 총급여 5,500만원 초과 시 13.2%, 이하는 16.5% 세액공제율.

4. **연금 수령 연령·기간.** 만 55세 이후, 가입 후 10년 이상 유지, 수령 기간 10년 이상 → **연금소득세 3.3-5.5%**. 조건 미충족 시 **기타소득세 16.5%** 중과되므로 무리한 중도인출 자제.

5. **국민연금 · 퇴직연금 · 개인연금 3층 구조.** 국민연금만으로는 은퇴 후 소득대체율 약 40% 수준. 퇴직연금+개인연금으로 70-80% 수준 목표가 현실적. 국민연금공단 [내연금 알아보기](https://www.nps.or.kr) 에서 예상수령액 조회 가능.

6. **TDF 국내 vs 미국 비교.** 한국 TDF는 미국 대비 **주식 비중이 전반적으로 낮고 (보수적) 보수율은 높음** (TER 0.5-0.9%). 공격형을 원하면 동일 빈티지 기준 미국 TDF 해외직투가 대안일 수 있으나 세제는 불리.
</KoreaContext>
