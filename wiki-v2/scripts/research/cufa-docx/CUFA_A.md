# CUFA 부록 A — 엑셀 사용법 가이드

재무모델 구축에 필요한 엑셀 기능을 다룬다. 엑셀을 처음 써보는 학생도 이 가이드를 따라하면 3-Statement 모델을 만들 수 있다.

# 1. 엑셀 기본

## 1-1. 셀과 수식

엑셀의 모든 것은 셀(Cell)에서 시작한다. 셀은 열(A, B, C...)과 행(1, 2, 3...)의 교차점이다. A1은 1행 A열의 셀.

수식은 = 으로 시작한다.

=A1+B1 → A1셀과 B1셀의 합.

=A1*B1 → 곱.

=A1/B1 → 나누기.

=A1^2 → 제곱.

셀을 클릭하면 수식 입력줄에 해당 셀의 내용(값 또는 수식)이 보인다.

## 1-2. 참조

상대 참조: =A1. 수식을 복사하면 참조도 같이 이동한다. A1을 아래로 복사하면 A2, A3...

절대 참조: =$A$1. 수식을 복사해도 참조가 고정된다. 핵심 가정(성장률, 세율 등)을 고정할 때 사용.

혼합 참조: =$A1 (열 고정) 또는 =A$1 (행 고정). 민감도 테이블에서 사용.

F4키: 셀 참조를 입력한 뒤 F4를 누르면 상대→절대→혼합→상대 순으로 전환된다. 가장 많이 쓰는 단축키 중 하나.

## 1-3. 색상 규칙 (업계 표준)

재무모델에서 색상은 데이터의 성격을 구분한다. 모든 증권사, 운용사에서 이 규칙을 따른다.

파란색 (Blue): 직접 입력한 값 (하드코딩). 가정, 외부 데이터.

검은색 (Black): 수식으로 계산된 값. 다른 셀을 참조하여 도출.

초록색 (Green): 다른 시트 또는 외부 파일에서 참조한 값.

빨간색 (Red): 체크 또는 경고. 수식 검증 시 사용.

이 규칙을 지키면 다른 사람이 모델을 열었을 때 "어디가 가정이고 어디가 수식인지" 즉시 파악할 수 있다. CUFA 보고서에서도 이 규칙을 적용한다.

## 1-4. 필수 단축키 Top 20

처음에는 Ctrl+C/V, F4, Ctrl+1, Alt+=만 익히면 된다. 나머지는 사용하면서 자연스럽게 외워진다.

# 2. 재무분석용 핵심 함수

## 2-1. SUM, AVERAGE, MIN, MAX

=SUM(A1:A5) → A1~A5의 합계.

=AVERAGE(A1:A5) → 평균.

=MIN(A1:A5) → 최솟값.

=MAX(A1:A5) → 최댓값.

재무 데이터의 합계(매출 합산), 평균(5개년 평균 OPM), 최대/최소(PER 밴드)에 사용.

## 2-2. IF

=IF(조건, 참일 때 값, 거짓일 때 값)

=IF(A1>0, "흑자", "적자") → A1이 양수면 "흑자", 아니면 "적자".

실전: =IF(영업이익>이자비용, "안전", "위험") → 이자보상배율 간이 체크.

## 2-3. VLOOKUP / INDEX-MATCH

VLOOKUP: 테이블에서 특정 값을 찾아서 대응하는 값을 반환.

=VLOOKUP(찾을값, 범위, 열번호, FALSE)

=VLOOKUP("SK하이닉스", A:D, 3, FALSE) → A열에서 "SK하이닉스"를 찾아 C열(3번째 열) 값을 반환.

INDEX-MATCH: VLOOKUP보다 유연하다. 왼쪽 열이 아닌 곳에서도 검색 가능.

=INDEX(반환할범위, MATCH(찾을값, 검색범위, 0))

실전: 여러 기업의 재무 데이터를 한 시트에 모아놓고, 기업명으로 검색하여 특정 비율을 자동으로 불러올 때 사용.

## 2-4. SUMIFS / COUNTIFS

조건부 합계/카운트. 여러 조건을 동시에 적용할 수 있다.

=SUMIFS(합산범위, 조건범위1, 조건1, 조건범위2, 조건2)

실전: 피어 비교 테이블에서 "동일 업종"이면서 "시총 3000억 이상"인 기업의 평균 PER을 구할 때.

## 2-5. 성장률 계산

YoY 성장률: =(당기-전기)/전기 → 셀 서식을 %로 설정.

CAGR: =(마지막값/첫번째값)^(1/기간수)-1

=($F2/$B2)^(1/4)-1 → 4년간 CAGR.

실전: 5개년 매출 CAGR 계산. 3주차 과제에서 바로 사용.

# 3. 재무모델 구축 실전

## 3-1. 시트 구조

3-Statement 모델의 시트 구성:

시트 1 — Assumptions (가정): 핵심 가정을 한곳에 모은다. 매출 성장률, GPM, CAPEX/매출, 세율 등. 모든 가정은 파란색. 이 시트의 숫자를 바꾸면 모델 전체가 자동 업데이트되는 구조를 만든다.

시트 2 — IS (손익계산서): 매출 → 매출원가 → 매출총이익 → 판관비 → 영업이익 → 이자비용 → 세전이익 → 법인세 → 순이익 → EPS.

시트 3 — CF (현금흐름): 순이익 + 감가상각비 +/- 운전자본 변동 = 영업CF. 영업CF - CAPEX = FCF.

시트 4 — BS (재무상태표): 전기말 BS + 당기 변동 = 당기말 BS.

시트 5 — Valuation: Peer Multiple + DCF + Football Field.

## 3-2. IS 시트 구축 step-by-step

열 구성: A열=항목명, B~F열=과거 5개년(2021~2025), G~H열=추정 2개년(2026E~2027E).

핵심: Assumptions 시트의 값(파란색)을 바꾸면 IS의 모든 추정치(검은색)가 자동 업데이트된다. 이 구조가 시나리오 분석의 기반이다.

## 3-3. CF 시트 구축

## 3-4. 민감도 테이블 (데이터 표 기능)

엑셀의 "데이터 표" 기능으로 2변수 민감도 테이블을 만든다.

1단계: 테이블의 왼쪽 열에 변수1의 값(예: WACC 8%, 9%, 10%, 11%, 12%).

2단계: 테이블의 상단 행에 변수2의 값(예: 영구성장률 1%, 1.5%, 2%, 2.5%, 3%).

3단계: 왼쪽 상단 교차점에 결과 셀(목표주가)을 참조하는 수식.

4단계: 테이블 전체 선택 → 데이터 탭 → "가상 분석" → "데이터 표" → 행 입력 셀(영구성장률 셀), 열 입력 셀(WACC 셀) 지정.

결과: 각 칸에 WACC/영구성장률 조합별 목표주가가 자동 계산된다. 6주차 DCF 민감도 테이블이 이것이다.

## 3-5. 차트 만들기

재무 분석에서 자주 쓰는 차트 3가지:

(1) 매출/영업이익 추이: 묶은 막대 차트. 매출은 넓은 막대, 영업이익은 좁은 막대 또는 꺾은선 중첩. 추정치는 패턴 채우기(빗금)로 구분.

(2) OPM/ROE 추이: 꺾은선 차트. 관심 기업은 굵은 실선, 피어는 가는 점선.

(3) Football Field: 가로 막대 차트. 각 방법(PER, DCF, PBR)별로 Bear~Bull 범위를 막대로 표시. 현재 주가를 세로 점선으로 표시.

차트 만들 때 반드시: 제목, 단위(억 원, 배, %), 출처(CompanyGuide, DART), 범례.

# 4. 피벗테이블

대량의 재무 데이터를 요약/분석할 때 강력한 도구다.

데이터 선택 → 삽입 → 피벗테이블.

행: 기업명.

열: 연도.

값: 매출(합계) 또는 OPM(평균).

이렇게 하면 여러 기업의 여러 연도 재무 데이터를 한눈에 볼 수 있는 테이블이 자동으로 만들어진다.

필터를 걸면 특정 업종, 특정 시총 범위의 기업만 볼 수도 있다. 4주차 피어 비교에서 유용하다.

# 5. 셀 서식과 조건부 서식

## 5-1. 숫자 형식 설정

Ctrl+1을 누르면 셀 서식 대화상자가 열린다.

재무모델에서 가장 많이 쓰는 형식:

숫자: 소수 자릿수 설정. 재무 비율은 소수 1자리(예: 8.2%), 금액은 정수.

백분율: %로 표시. 0.082를 입력하면 8.2%로 보여줌.

사용자 지정 형식:

#,##0 → 천 단위 콤마, 정수. 매출/이익 표시에 사용.

#,##0.0 → 소수 1자리.

0.0% → 백분율 소수 1자리.

#,##0;[Red]-#,##0 → 양수는 검은색, 음수는 빨간색 자동 표시.

#,##0, → 천 단위 절사 (1,000 → 1). 대형 기업 데이터를 억 단위로 표시할 때.

팁: 재무 데이터의 단위를 시트 상단에 반드시 명시한다. "(단위: 억 원)" 또는 "(단위: 백만 원)". 단위가 섞이면 결과가 1000배 차이 나는 대참사가 발생한다.

## 5-2. 조건부 서식

홈 탭 → 조건부 서식에서 설정한다.

실전 활용 1: OPM 비교.

OPM이 10% 이상이면 초록색 배경, 5% 이하면 빨간색 배경. 피어 비교 테이블에서 시각적으로 강약을 한눈에 파악할 수 있다.

실전 활용 2: YoY 성장률.

양수면 초록 글씨, 음수면 빨간 글씨. 5개년 추이에서 성장/감소를 즉시 식별.

실전 활용 3: 데이터 막대.

셀 안에 작은 막대 차트를 넣어서 크기를 시각화. 매출 규모 비교에 유용.

실전 활용 4: 민감도 히트맵.

민감도 테이블에 색조(Color Scale) 적용. 높은 값=진한 초록, 낮은 값=진한 빨강. 어떤 가정 조합이 유리한지 시각적으로 파악.

## 5-3. 오류 처리

=IFERROR(수식, 대체값) → 수식이 오류를 반환하면 대체값을 표시한다.

=IFERROR(A1/B1, 0) → B1이 0이면 #DIV/0! 대신 0 표시.

=IFERROR(VLOOKUP("삼성전자",A:D,3,FALSE), "없음") → 찾는 값이 없으면 "없음" 표시.

재무모델에서 IFERROR는 필수다. 적자 기업의 PER(음수 순이익으로 나누기) 같은 경우에 오류가 나오기 때문.

# 7. 차트 만들기 상세

## 7-1. 매출/이익 추이 차트

가장 기본적인 재무 차트다.

데이터: 연도(X축), 매출(막대, 왼쪽Y축), OPM(꺾은선, 오른쪽Y축).

만드는 법:

1. 연도, 매출 데이터 선택 → 삽입 → 묶은 세로 막대.

2. OPM 데이터 추가: 차트 클릭 → 데이터 선택 → 계열 추가.

3. OPM 계열 우클릭 → 계열 차트 종류 변경 → 꺾은선.

4. OPM 계열 우클릭 → 보조 축으로 그리기.

5. 추정치(2026E~) 막대는 다른 색 또는 빗금 패턴.

서식:

제목: "코스맥스 매출 및 OPM 추이"

왼쪽 Y축: (억 원)

오른쪽 Y축: (%)

출처: "출처: DART, CUFA 추정"

## 7-2. 피어 비교 막대 차트

데이터: 기업명(X축), 비율값(Y축). 관심기업 = 보라색, 피어 = 회색.

만드는 법:

1. 기업명, OPM 데이터 선택 → 삽입 → 묶은 세로 막대.

2. 관심기업 막대만 선택(더블클릭) → 채우기 색상을 #7c6af7(CUFA 보라)으로.

3. 피어 막대 = #999999(회색).

이 차트 하나로 "관심 기업이 피어 대비 어디에 위치하는가"가 시각적으로 드러난다.

## 7-3. Football Field 차트

가장 어렵지만 가장 인상적인 차트.

데이터 준비:

원래 데이터가 Bear/Base/Bull이면, 차트용으로 변환해야 한다.

만드는 법:

1. 위 데이터 선택 → 삽입 → 누적 가로 막대.

2. "투명" 계열 선택 → 채우기 없음, 테두리 없음.

3. "Bear~Base" = 노란색.

4. "Base~Bull" = 초록색.

5. 현재 주가를 세로 점선으로 추가(도형 삽입).

이 차트가 보고서 밸류에이션 섹션의 핵심 시각자료다.

## 9-1. 흔한 오류 Top 5

(1) #DIV/0! → 0으로 나눔. IFERROR로 감싸거나 분모 확인.

(2) #REF! → 참조 셀 삭제됨. 행/열 삭제 시 수식 확인. Ctrl+Z로 복구.

(3) #VALUE! → 숫자와 텍스트 혼재. VALUE() 함수로 변환.

(4) 순환 참조 → IS→CF→BS→IS로 돌아가는 경우. 이자비용이 부채에 의존하고, 부채가 CF에 의존하고, CF가 IS에 의존하면 순환 발생. 초보 단계에서는 이자비용을 고정값으로 넣어 순환을 끊는다.

(5) 단위 불일치 → 억 원과 원이 섞이면 결과가 1억 배 차이. 시트 상단에 단위를 명시하고, 모든 셀의 단위를 통일한다.

# 10. 실전 팁 모음

## 10-1. 데이터 입력 속도 올리기

CompanyGuide에서 복사: 웹 테이블 선택 → Ctrl+C → 엑셀에 Ctrl+Shift+V (값만 붙여넣기). 서식 없이 숫자만 들어간다.

네이버 금융 엑셀 다운로드: "재무분석" 탭 → "엑셀 다운로드" 버튼. 5개년 데이터가 한번에 나온다.

## 10-2. 이름 정의 (Named Range)

수식 탭 → 이름 관리자에서 셀에 이름을 부여한다.

Assumptions!$C$2를 "매출성장률"로 정의하면, IS에서 =D2*(1+매출성장률)로 쓸 수 있다.

핵심 가정 5~10개에 이름을 부여하면 수식의 가독성이 크게 올라간다.

## 10-3. 인쇄 설정

보고서 Appendix에 엑셀 테이블을 넣을 때: 페이지 레이아웃 → 인쇄 영역 설정 → PDF로 내보내기.

인쇄 시 열이 잘리지 않게: 페이지 레이아웃 → 크기 조정 → 너비: 1페이지.

## 10-4. 엑셀 vs Google Sheets

엑셀 장점: 데이터 표(민감도), 복잡한 차트, VBA 매크로, 대용량 데이터.

Google Sheets 장점: 실시간 협업, 자동 저장, 무료.

CUFA 과제는 엑셀을 권장한다. 데이터 표 기능(민감도 테이블)이 Google Sheets에는 없기 때문.

다만 팀 프로젝트에서 실시간 공동작업이 필요하면 Google Sheets를 쓰고, 최종본을 엑셀로 변환하는 방법도 있다.

## 10-5. 자주 쓰는 추가 함수 정리

=ROUND(A1, 1) → 소수 1자리 반올림.

=ABS(A1) → 절대값. 음수를 양수로.

=YEAR(A1), =MONTH(A1) → 날짜에서 연/월 추출.

=TEXT(A1, "#,##0") → 숫자를 텍스트로 변환. 보고서 문장에 숫자를 넣을 때.

=CONCATENATE(A1, " ", B1) 또는 =A1&" "&B1 → 텍스트 연결.

=RANK(A1, A:A) → 순위. 피어 비교에서 비율 순위 매기기.

=PERCENTILE(범위, 0.25) → 사분위수. 분포 분석.
