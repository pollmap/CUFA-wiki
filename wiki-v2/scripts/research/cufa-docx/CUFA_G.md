# CUFA 부록 G — 전체 과제 체크리스트 + 보고서 템플릿

8주 과제를 한눈에 볼 수 있는 체크리스트와, 7주차에 사용할 보고서 보일러플레이트를 통합 정리한다.

# 1. 8주 과제 전체 체크리스트

각 과제가 최종 보고서의 어느 섹션에 대응되는지 표시한다.

필수 과제만 성실히 하면, 7주차에 보고서의 90%가 이미 준비된 상태다.

# 2. 주차별 과제 상세

## W1. 관심 기업 선정

조건: 시총 3,000억~3조, 코스피/코스닥 상장, 금융/지주 제외, 증권사 리포트 2편 이상 존재.

산출물: 아래 양식을 채운다.

```

기업명: __________ (종목코드: ______)

시가총액: ____억 원

업종(GICS): __________

선정 이유 (3줄): __________

```

## W2. 산업 브리핑

1p에 아래 항목을 포함한다.

```

산업명: __________

시장 규모: 글로벌 ____억 달러, 한국 ____억 원

CAGR (과거 5년 / 향후 5년): ___% / ___%

경쟁 구도: 1위 ____(M/S __%), 2위 ____(M/S __%)

산업생명주기: 도입 / 성장 / 조정 / 성숙 / 쇠퇴

핵심 성장 드라이버: __________

핵심 리스크: __________

```

## W3~W4. 재무 분석

엑셀에 아래 항목을 5개년(과거 3년 + 추정 2년)으로 정리한다.

IS: 매출, 매출원가, 매출총이익, 판관비, 영업이익, 이자비용, 세전이익, 법인세, 순이익.

CF: 영업CF, CAPEX, FCF.

BS: 현금, 매출채권, 재고, 유형자산, 차입금, 자본총계.

비율: GPM, OPM, NPM, ROE, 부채비율, 이자보상배율, CCC.

피어 비교: 관심기업 vs Peer A vs Peer B (3개년 비율 비교).

## W5. 실적 추정

3-Statement 모델에 Bull/Base/Bear 3시나리오를 적용.

확률 가중:    25%     50%     25%

기대 EPS:    ___원

```

```
