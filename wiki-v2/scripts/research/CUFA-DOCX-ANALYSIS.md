## 7. v2 wiki 통합 옵션 (찬희 결정 필요)

저작권 명시 ("외부 유출·SNS 게시·타 학회 공유 금지") + "위반 시 민·형사 책임" 표기 때문에 **그대로 공개 wiki 게시 X**. 4개 옵션:

### Option A — 전체 공개 + 저작권 박스
- 16편 전체 공개 + 모든 페이지 상단에 저작권 박스 (CC-BY-NC 라이선스로 명시 권장)
- 외부 인용 시 출처 표기 의무
- **장점**: 콘텐츠 곧바로 활용 / SEO 효과 / CUFA 인지도
- **단점**: 학회원 권리 보존 X / 무단 재배포 통제 어려움

### Option B — 학회원 로그인 영역만 공개
- v2 wiki에 NextAuth 추가 + 학회원 GitHub OAuth (또는 학번 검증)
- 게스트는 목차·요약만, 본문은 로그인 후
- **장점**: 권리 보존 / CUFA 가입 인센티브
- **단점**: Phase 4 인증 시스템 필수 / 외부 SEO 효과 X

### Option C — 공개판 = 요약·발췌, 전체는 학회원 only ⭐ 추천
- 공개 wiki에 각 주차 **300~500자 요약 + 핵심 차트 1장** + "전체 보기 (학회원)" 링크
- 학회원 영역에 16편 전체
- **장점**: SEO + 저작권 보호 + 가입 유인 동시
- **단점**: 요약 별도 작성 필요 (1주 인력)

### Option D — wiki에 안 올림 (학회 내부 PDF만)
- 16편은 학회 슬랙/노션 등 내부에 PDF로 유통
- v2 wiki는 별도 공개 콘텐츠로 작성
- **장점**: 분리 명확
- **단점**: 콘텐츠 자산 낭비
