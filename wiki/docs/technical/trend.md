# 추세 분석

> "The trend is your friend." (추세는 당신의 친구)

## 추세의 정의

### 추세의 3가지 방향

```
상승추세 (Uptrend)
        ╱╲
       ╱  ╲╱╲
      ╱      ╲╱╲
     ╱          ╲
    ╱
   └─ Higher Highs, Higher Lows

하락추세 (Downtrend)
    ╲
     ╲╱╲
        ╲╱╲
           ╲╱╲
              ╲
   └─ Lower Highs, Lower Lows

횡보 (Sideways/Range)
   ───────────────
        ╱╲  ╱╲
   ╲  ╱    ╲╱  ╲
    ╲╱          ╲
   ───────────────
   └─ 고점과 저점이 일정 범위
```

### 추세의 3단계 (다우 이론)

```
1차 추세 (Primary Trend)
├── 1년 이상 지속
├── 장기 투자자용
└── 강세장/약세장 구분

2차 추세 (Secondary Trend)
├── 3주 ~ 3개월
├── 1차 추세의 조정
└── 스윙 트레이더용

3차 추세 (Minor Trend)
├── 3주 미만
├── 2차 추세 내 단기 파동
└── 단기 트레이더용
```

## 추세선 (Trend Line)

### 상승 추세선

```
        ╱
       ╱╲ ← 고점 3
      ╱  ╲╱
     ╱    ╲╱╲ ← 고점 2
    ╱        ╲╱
   ╱     저점 3 ╲╱╲ ← 고점 1
  ╱           저점 2 ╲╱
 ╱                저점 1
─────────────────────────
       ↑
    추세선 (저점 연결)
```

**작도 규칙:**
1. 최소 2개의 저점 연결
2. 3개 이상이면 신뢰도 상승
3. 가격이 추세선 터치 횟수가 많을수록 강함

### 하락 추세선

```
 ╲
  ╲╱╲ 고점 1
     ╲╱
      ╲╱╲ 고점 2
         ╲╱
          ╲╱╲ 고점 3
             ╲╱
              ╲
─────────────────────────
       ↑
    추세선 (고점 연결)
```

### 추세선 유효성

| 요소 | 강한 추세선 | 약한 추세선 |
|------|------------|------------|
| 터치 횟수 | 3회 이상 | 2회 |
| 기간 | 장기간 | 단기간 |
| 각도 | 30-45도 | 급경사/완경사 |
| 돌파 | 거래량 동반 | 거래량 미미 |

## 추세 채널

### 채널 작도

```
   상단선 (저항) ───────────────────
              ╱╲      ╱╲
             ╱  ╲    ╱  ╲
            ╱    ╲  ╱    ╲
           ╱      ╲╱      ╲
          ╱                ╲
   하단선 (지지) ───────────────────

상승 채널: 평행한 두 상향 추세선
하락 채널: 평행한 두 하향 추세선
```

### 채널 매매 전략

```
상승 채널 내:
- 하단선 터치 → 매수
- 상단선 터치 → 일부 매도/관망
- 하단선 붕괴 → 손절
- 상단선 돌파 → 강세 가속

하락 채널 내:
- 상단선 터치 → 매도/공매도
- 하단선 터치 → 일부 청산
- 상단선 돌파 → 손절
- 하단선 붕괴 → 약세 가속
```

## 지지와 저항

### 수평 지지/저항

```
   저항선 ─────┬─────┬─────
              │     │
        ╱╲    │╱╲   │
       ╱  ╲   ╱  ╲  │
      ╱    ╲ ╱    ╲ │
     ╱      X      ╲│
   지지선 ─────┴─────┴─────

지지: 하락을 멈추게 하는 가격대
저항: 상승을 멈추게 하는 가격대
```

### 역할 전환 (Role Reversal)

```
저항 돌파 → 지지로 전환

   ──────────────────────────
   구 저항선 = 신 지지선
        │        ╱
        │  ╱╲   ╱
        │╱  ╲ ╱   ← 되돌림 후 반등
        X    X
       ╱
      ╱
     ╱
```

### 강한 지지/저항 조건

```
□ 여러 번 테스트된 레벨
□ 높은 거래량 동반
□ 라운드 넘버 (만원 단위 등)
□ 과거 고점/저점
□ 이동평균선과 일치
□ 피보나치 레벨과 일치
```

## 추세 전환 신호

### 추세 전환 패턴

```
1. 헤드앤숄더 (상승→하락)

        고점2
         ╱╲
   고점1╱  ╲고점3
    ╱╲╱    ╲╱╲
   ╱          ╲
  ──────────────── 넥라인

2. 역헤드앤숄더 (하락→상승)

  ──────────────── 넥라인
   ╲          ╱
    ╲╱╲    ╱╲╱
   저점1╲  ╱저점3
         ╲╱
        저점2
```

### 추세 전환 확인

```
상승→하락 전환:
1. Higher High 실패 (이전 고점 돌파 실패)
2. Lower Low 형성 (이전 저점 하회)
3. 추세선 붕괴
4. 이동평균선 데드크로스
5. 거래량 급증 (매도)

하락→상승 전환:
1. Lower Low 실패 (이전 저점 방어)
2. Higher High 형성 (이전 고점 돌파)
3. 추세선 돌파
4. 이동평균선 골든크로스
5. 거래량 급증 (매수)
```

## 피보나치 되돌림

### 주요 레벨

```
   100% ────────────── 고점
         │
    78.6% ─ ─ ─ ─ ─ ─
         │
    61.8% ─ ─ ─ ─ ─ ─ (황금비)
         │
    50.0% ─ ─ ─ ─ ─ ─
         │
    38.2% ─ ─ ─ ─ ─ ─ (일반적 되돌림)
         │
    23.6% ─ ─ ─ ─ ─ ─
         │
     0% ────────────── 저점
```

### 활용법

```
상승 후 조정:
- 38.2% 되돌림: 강한 추세
- 50% 되돌림: 보통 추세
- 61.8% 되돌림: 약한 추세 (마지막 지지)
- 61.8% 이탈: 추세 전환 가능성

진입 전략:
- 38.2%, 50%, 61.8%에서 지지 확인 후 매수
- 손절: 다음 피보나치 레벨 아래
```

## 추세 강도 측정

### ADX (Average Directional Index)

```
ADX 값과 추세 강도:
0-20:  추세 없음/약함
20-40: 추세 형성 중
40-60: 강한 추세
60+:   매우 강한 추세

매매 전략:
- ADX > 25: 추세 추종 전략
- ADX < 20: 횡보/역추세 전략
```

### 이격도

```
이격도 = (현재가 / 이동평균) × 100

상승추세:
- 105% 이상: 과열 (단기 조정 가능)
- 100-105%: 정상 범위

하락추세:
- 95% 이하: 과매도 (반등 가능)
- 95-100%: 정상 범위
```

## 실전 추세 매매

### 추세 매매 원칙

```
1. 추세 확인
   └── 상위 시간대에서 방향 확인

2. 조정 대기
   └── 지지선/추세선까지 눌림 대기

3. 진입
   └── 반전 캔들 + 거래량 확인 후

4. 손절
   └── 추세선 이탈 시

5. 청산
   └── 추세 전환 신호 시
```

### 체크리스트

```
진입 전:
□ 주 추세 방향 확인 (상위 시간대)
□ 조정 완료 여부 (피보나치, 추세선)
□ 반전 신호 확인 (캔들, 지표)
□ 손절 가격 설정
□ 목표가 설정

보유 중:
□ 추세선 유지 여부
□ 고점/저점 패턴 유지
□ 지표 다이버전스 주의
□ 트레일링 스탑 조정
```
