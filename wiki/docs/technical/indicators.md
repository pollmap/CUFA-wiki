# 기술적 지표

## RSI (Relative Strength Index)

### 개념

**상대강도지수(RSI)** 는 일정 기간 동안 상승폭과 하락폭을 비교하여 과매수/과매도 상태를 판단하는 모멘텀 지표입니다.

### 공식

```
RSI = 100 - (100 / (1 + RS))

RS = 평균 상승폭 / 평균 하락폭 (n일간)

일반적으로 n = 14일 사용
```

### RSI 해석

```
  100 ─────────────────────
       과매수 (70 이상)
   70 ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─

       중립

   30 ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─
       과매도 (30 이하)
    0 ─────────────────────
```

| RSI 값 | 상태 | 전략 |
|--------|------|------|
| 70+ | 과매수 | 매도 고려 |
| 50-70 | 강세 | 보유/매수 |
| 30-50 | 약세 | 관망/매도 |
| 30- | 과매도 | 매수 고려 |

### RSI 다이버전스

```
강세 다이버전스 (Bullish):
가격: 저점 ↘ 더 낮은 저점
RSI:  저점 ↗ 더 높은 저점
→ 하락세 약화, 상승 반전 가능

약세 다이버전스 (Bearish):
가격: 고점 ↗ 더 높은 고점
RSI:  고점 ↘ 더 낮은 고점
→ 상승세 약화, 하락 반전 가능
```

### RSI 활용 팁

```
□ 추세장: 과매수/과매도 수준 조정
   - 상승추세: 40-80
   - 하락추세: 20-60

□ 50선 돌파
   - 50 상향 돌파 → 강세 전환
   - 50 하향 돌파 → 약세 전환

□ Failure Swing
   - RSI가 70 초과 후 하락
   - 반등하나 70 못 넘김
   - 이전 저점 하회 → 강한 매도 신호
```

## MACD (Moving Average Convergence Divergence)

### 개념

**MACD**는 두 이동평균선의 수렴과 발산을 나타내는 추세 추종 모멘텀 지표입니다.

### 구성 요소

```
MACD Line = 12일 EMA - 26일 EMA
Signal Line = MACD의 9일 EMA
Histogram = MACD Line - Signal Line
```

### MACD 차트

```
      │
      │  ╱╲
  ────┼──────────── 0선
      │╲  ╱
      │ ╲╱
      │

MACD Line (실선)
Signal Line (점선)
Histogram (막대)
```

### MACD 신호

| 신호 | 조건 | 의미 |
|------|------|------|
| 매수 | MACD가 Signal 상향 돌파 | 상승 모멘텀 |
| 매도 | MACD가 Signal 하향 돌파 | 하락 모멘텀 |
| 강세 | MACD가 0선 위 | 상승 추세 |
| 약세 | MACD가 0선 아래 | 하락 추세 |

### MACD 다이버전스

```
강세 다이버전스:
가격: 신저가 형성
MACD: 이전 저점보다 높은 저점
→ 상승 반전 예상

약세 다이버전스:
가격: 신고가 형성
MACD: 이전 고점보다 낮은 고점
→ 하락 반전 예상
```

### MACD 활용 전략

```
1. 기본 크로스 전략
   - 매수: MACD > Signal (골든크로스)
   - 매도: MACD < Signal (데드크로스)

2. 제로선 크로스
   - 매수: MACD가 0선 상향 돌파
   - 매도: MACD가 0선 하향 돌파

3. 히스토그램 분석
   - 막대 증가: 모멘텀 강화
   - 막대 감소: 모멘텀 약화
```

## 스토캐스틱 (Stochastic)

### 개념

**스토캐스틱**은 현재 가격이 일정 기간의 가격 범위에서 어디에 위치하는지 보여주는 모멘텀 지표입니다.

### 공식

```
%K = (현재가 - n일 최저가) / (n일 최고가 - n일 최저가) × 100
%D = %K의 m일 이동평균

일반 설정: (14, 3, 3)
- n = 14일
- %K 평활화 = 3일
- %D = 3일
```

### 스토캐스틱 해석

```
  100 ─────────────────────
       과매수 (80 이상)
   80 ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─

       │╱╲  ← %K (빠른선)
       ╱  ╲ ← %D (느린선)

   20 ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─
       과매도 (20 이하)
    0 ─────────────────────
```

### 스토캐스틱 신호

| 신호 | 조건 | 전략 |
|------|------|------|
| 매수 | %K가 %D 상향 돌파 (20 이하에서) | 진입 |
| 매도 | %K가 %D 하향 돌파 (80 이상에서) | 청산 |
| 강세 | 두 선 모두 50 위 | 상승 추세 |
| 약세 | 두 선 모두 50 아래 | 하락 추세 |

### 스토캐스틱 vs RSI

| 구분 | 스토캐스틱 | RSI |
|------|-----------|-----|
| 민감도 | 높음 | 중간 |
| 신호 빈도 | 많음 | 적음 |
| 노이즈 | 많음 | 적음 |
| 적합 시장 | 횡보장 | 추세장 |

## 지표 조합 전략

### RSI + MACD 조합

```
강한 매수 신호:
1. RSI가 30 아래에서 30 위로
2. MACD 골든크로스 동시 발생
3. 거래량 증가

강한 매도 신호:
1. RSI가 70 위에서 70 아래로
2. MACD 데드크로스 동시 발생
3. 거래량 증가
```

### 멀티 타임프레임 분석

```
장기 (주봉): 추세 방향 확인
  │
  ├─ MACD 방향 확인
  │
중기 (일봉): 진입 시점 결정
  │
  ├─ RSI 과매수/과매도
  │
단기 (4시간): 정밀 타이밍
  │
  └─ 스토캐스틱 크로스
```

## 지표의 한계

### 공통 문제점

```
1. 후행성
   - 지표는 과거 데이터 기반
   - 추세 전환 후에 신호

2. 거짓 신호
   - 특히 횡보장에서 빈번
   - 손절 필수

3. 파라미터 최적화 함정
   - 과거에 맞춘 설정
   - 미래에는 다를 수 있음
```

### 해결 방안

```
□ 여러 지표 교차 확인
□ 가격 패턴과 병행
□ 거래량 확인
□ 지지/저항 레벨 참고
□ 손절선 설정 필수
□ 백테스트로 검증
```

## 실전 체크리스트

```
매수 진입 전:
□ 상위 시간대 추세 확인 (MACD)
□ RSI 과매도 or 50선 돌파
□ 스토캐스틱 골든크로스
□ 지지선 근처
□ 거래량 증가
□ 손절선 설정

매도 청산 전:
□ RSI 과매수 or 다이버전스
□ MACD 데드크로스
□ 스토캐스틱 80 위에서 하락
□ 저항선 근처
□ 거래량 급증 (클라이맥스)
```
