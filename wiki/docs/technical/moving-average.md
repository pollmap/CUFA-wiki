# 이동평균선 (Moving Average)

## 이동평균선이란?

**이동평균선 (MA)** 은 일정 기간 동안의 가격 평균을 연결한 선으로, 추세를 파악하는 가장 기본적인 기술적 지표입니다.

## 이동평균선 종류

### 1. 단순이동평균 (SMA)

```
SMA = (P1 + P2 + ... + Pn) / n

예: 5일 SMA
일자:  1    2    3    4    5
가격: 100  102  101  103  104

5일 SMA = (100 + 102 + 101 + 103 + 104) / 5 = 102
```

**특징:**
- 계산이 간단
- 모든 데이터에 동일 가중치
- 후행성 있음

### 2. 지수이동평균 (EMA)

```
EMA = 오늘 가격 × k + 어제 EMA × (1-k)
k = 2 / (n + 1)

예: 5일 EMA의 k = 2/(5+1) = 0.333

최근 가격에 더 높은 가중치 부여
```

**특징:**
- 최근 가격 반영 빠름
- SMA보다 민감
- 단기 트레이딩에 유리

### SMA vs EMA 비교

```
         가격
          │     ╱── 가격
          │   ╱╲╱
          │  ╱    ── EMA (더 민감)
          │ ╱   ╱
          │╱  ╱── SMA (더 평탄)
          └────────────────────
                시간
```

| 구분 | SMA | EMA |
|------|-----|-----|
| 민감도 | 낮음 | 높음 |
| 노이즈 | 적음 | 많음 |
| 지연 | 큼 | 적음 |
| 용도 | 장기 추세 | 단기 추세 |

## 주요 이동평균 기간

### 일반적인 기간 설정

| 기간 | 용도 | 특징 |
|------|------|------|
| 5일 | 초단기 | 1주일 평균 |
| 10일 | 단기 | 2주 평균 |
| 20일 | 단기-중기 | 1개월 평균 |
| 50일 | 중기 | 약 2.5개월 |
| 60일 | 중기 | 분기 평균 |
| 120일 | 중장기 | 반기 평균 |
| 200일 | 장기 | 약 1년 |

### 한국 시장에서 자주 사용

```
단기: 5일, 10일, 20일
중기: 60일, 120일
장기: 200일, 250일 (1년 영업일)
```

## 이동평균선 해석

### 추세 판단

```
상승추세:
가격 > 단기MA > 중기MA > 장기MA
        (정배열)

하락추세:
가격 < 단기MA < 중기MA < 장기MA
        (역배열)
```

### 지지와 저항

```
상승추세에서:
       │    ╱
       │  ╱    ← 가격
       │╱
  ─────┼───── ← MA (지지선 역할)
       │

MA에 닿으면 반등하는 경향
```

```
하락추세에서:
       │
  ─────┼───── ← MA (저항선 역할)
       │╲
       │  ╲   ← 가격
       │    ╲

MA에 닿으면 하락하는 경향
```

## 크로스 전략

### 골든크로스 (Golden Cross)

```
매수 신호

       │         ╱─ 단기MA
       │       ╱
       │     ╱
  ─────┼───X──────── 장기MA
       │ ╱
       │╱

단기MA가 장기MA를 상향 돌파
→ 상승 추세 전환 신호
```

### 데드크로스 (Dead Cross)

```
매도 신호

       │╲
       │ ╲
  ─────┼───X──────── 장기MA
       │     ╲
       │       ╲
       │         ╲─ 단기MA

단기MA가 장기MA를 하향 돌파
→ 하락 추세 전환 신호
```

### 크로스 조합

| 조합 | 용도 | 특징 |
|------|------|------|
| 5일 & 20일 | 단기 | 신호 빈번, 노이즈 많음 |
| 20일 & 60일 | 중기 | 적절한 균형 |
| 50일 & 200일 | 장기 | 신호 적음, 신뢰도 높음 |

## 그랜빌의 8법칙

Joseph Granville이 제시한 매매 법칙:

### 매수 신호 (4가지)

```
1. MA 하향 → 횡보 → 상승 전환
   가격이 MA 상향 돌파

2. 상승 중인 MA를
   가격이 잠시 하회 후 재상승

3. 상승 중인 MA 위에서
   가격 하락 → MA 닿기 전 반등

4. 가격이 하락 중인 MA 대폭 하회
   (MA 복귀 기대 - 고위험)
```

### 매도 신호 (4가지)

```
5. MA 상향 → 횡보 → 하락 전환
   가격이 MA 하향 돌파

6. 하락 중인 MA를
   가격이 잠시 상회 후 재하락

7. 하락 중인 MA 아래에서
   가격 상승 → MA 닿기 전 하락

8. 가격이 상승 중인 MA 대폭 상회
   (MA 복귀 기대 - 고위험)
```

## 다중 이동평균 전략

### 3중 MA 시스템

```
설정: 단기 5일, 중기 20일, 장기 60일

강한 매수:
5일 > 20일 > 60일 (정배열)
+ 가격이 모든 MA 위

강한 매도:
5일 < 20일 < 60일 (역배열)
+ 가격이 모든 MA 아래
```

### MA 밴드

```
         상단 밴드 (+2%)
  ───────────────────────
            │
    ────────MA────────
            │
  ───────────────────────
         하단 밴드 (-2%)

하단 밴드 → 과매도 → 매수 고려
상단 밴드 → 과매수 → 매도 고려
```

## 이동평균선의 한계

### 주의점

```
1. 후행 지표
   - 추세 확인에는 유용
   - 추세 전환 예측에는 늦음

2. 횡보장에서 비효율
   - 잦은 거짓 신호
   - 손실 누적 가능

3. 갭 발생 시
   - 크로스 신호 왜곡
   - 실제 추세와 괴리
```

### 해결 방법

```
□ 다른 지표와 병행 (RSI, MACD)
□ 거래량 확인
□ 지지/저항 레벨 참고
□ 멀티 타임프레임 분석
□ 필터 규칙 추가 (X% 돌파 시만)
```

## 실전 설정 예시

### 스윙 트레이딩

```
차트: 일봉
MA: 20일 EMA, 50일 SMA

진입:
- 가격이 20 EMA 위
- 20 EMA가 50 SMA 위
- 가격이 20 EMA로 눌림 후 반등

청산:
- 20 EMA 하향 돌파
- 또는 수익 목표 달성
```

### 장기 투자

```
차트: 주봉
MA: 10주 SMA, 40주 SMA

진입:
- 10주 MA가 40주 MA 골든크로스
- 확인 후 분할 매수

청산:
- 10주 MA가 40주 MA 데드크로스
- 또는 펀더멘털 변화
```
