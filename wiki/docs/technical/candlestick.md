# 캔들차트 패턴

## 캔들차트 기본 구조

### 양봉과 음봉

```
      양봉 (상승)              음봉 (하락)

         │ ← 윗꼬리            │ ← 윗꼬리
      ┌──┴──┐ ← 고가        ┌──┴──┐ ← 고가
      │     │              │█████│
      │     │ ← 몸통       │█████│ ← 몸통
      │     │              │█████│
      └──┬──┘ ← 저가        └──┬──┘ ← 저가
         │ ← 아래꼬리          │ ← 아래꼬리

  시가 < 종가           시가 > 종가
  (흰색/녹색)           (검정/빨강)
```

### 캔들 해석

| 요소 | 의미 |
|------|------|
| 긴 몸통 | 강한 매수/매도 압력 |
| 짧은 몸통 | 약한 움직임, 불확실성 |
| 긴 윗꼬리 | 고점에서 매도 압력 |
| 긴 아래꼬리 | 저점에서 매수 지지 |
| 꼬리 없음 | 강한 추세 |

## 단일 캔들 패턴

### 도지 (Doji)

시가와 종가가 거의 같은 캔들입니다.

```
   │           │             │
   ┼      ─────┼─────    ────┼
   │           │             │

 기본 도지   긴다리 도지    잠자리 도지
 (불확실)    (전환 가능)    (상승 반전)

                               │
                          ─────┼

                          비석 도지
                         (하락 반전)
```

**해석:**
- 추세 전환 가능성
- 강한 추세 후 출현 시 주의
- 확인 캔들 필요

### 망치형 (Hammer)

```
상승 반전 신호

      │        ← 작은 몸통 (상단)
   ┌──┴──┐
   └──┬──┘
      │        ← 긴 아래꼬리 (몸통의 2배+)
      │
      │

하락추세 끝에 나타남
→ 저점에서 매수세 유입
```

### 교수형 (Hanging Man)

```
하락 반전 신호

      │        ← 작은 몸통 (상단)
   ┌──┴──┐
   └──┬──┘
      │        ← 긴 아래꼬리
      │
      │

상승추세 끝에 나타남
→ 매도 압력 시작
```

### 슈팅스타 (Shooting Star)

```
하락 반전 신호

      │        ← 긴 윗꼬리 (몸통의 2배+)
      │
      │
   ┌──┴──┐
   └─────┘     ← 작은 몸통 (하단)

상승추세 끝에 나타남
→ 고점에서 매도 압력
```

### 역망치형 (Inverted Hammer)

```
상승 반전 신호

      │        ← 긴 윗꼬리
      │
      │
   ┌──┴──┐
   └─────┘     ← 작은 몸통 (하단)

하락추세 끝에 나타남
→ 매수 시도 시작
```

### 마루보즈 (Marubozu)

```
양봉 마루보즈          음봉 마루보즈

┌─────────┐         ┌█████████┐
│         │         │█████████│
│         │         │█████████│
│         │         │█████████│
└─────────┘         └█████████┘

(꼬리 없음)          (꼬리 없음)
강한 상승세          강한 하락세
```

## 2개 캔들 패턴

### 장악형 (Engulfing)

```
상승 장악형 (Bullish)           하락 장악형 (Bearish)

    ┌███┐                           ┌───┐
    │███│    ← 음봉              ← │   │ 양봉
    └███┘                           └───┘
┌─────────┐  ← 양봉             ┌█████████┐
│         │    (더 큼)          │█████████│ ← 음봉
└─────────┘                     └█████████┘   (더 큼)

하락 후 → 상승 반전          상승 후 → 하락 반전
```

**신뢰도 높은 조건:**
- 두 번째 캔들이 첫 번째를 완전히 감쌈
- 거래량 증가 동반
- 지지/저항 근처에서 출현

### 관통형 (Piercing Line)

```
상승 반전 신호

    ┌███┐
    │███│  ← 긴 음봉
    │███│
    └███┘
    ┌───┐
    │   │  ← 양봉 (음봉 중간 이상 관통)
    │   │
└───┴───┘
↑
갭 다운 시작
```

### 먹구름형 (Dark Cloud Cover)

```
하락 반전 신호

    ┌───┐
    │   │  ← 긴 양봉
    │   │
    └───┘
    ┌███┐
    │███│  ← 음봉 (양봉 중간 이하로 하락)
    │███│
└───┴███┘
↑
갭 업 시작
```

### 하라미 (Harami)

```
하라미 (임신)

강세 하라미              약세 하라미

┌█████████┐             ┌─────────┐
│█████████│             │         │
│  ┌───┐  │             │  ┌███┐  │
│  │   │  │             │  │███│  │
│  └───┘  │             │  └███┘  │
└█████████┘             └─────────┘

음봉 안에 양봉           양봉 안에 음봉
→ 상승 반전 가능         → 하락 반전 가능
```

## 3개 캔들 패턴

### 샛별형 (Morning Star)

```
강력한 상승 반전 신호

    ┌███┐
    │███│
    │███│      ← 긴 음봉
    └███┘
       ┌┐     ← 작은 몸통 (도지 가능)
       └┘       갭 다운
    ┌─────┐
    │     │   ← 긴 양봉
    │     │     (음봉의 중간 이상)
    └─────┘
```

### 저녁별형 (Evening Star)

```
강력한 하락 반전 신호

    ┌───┐
    │   │
    │   │      ← 긴 양봉
    └───┘
       ┌┐     ← 작은 몸통 (도지 가능)
       └┘       갭 업
    ┌█████┐
    │█████│   ← 긴 음봉
    │█████│     (양봉의 중간 이하)
    └█████┘
```

### 세 병사 (Three White Soldiers)

```
강세 지속 신호

        ┌───┐
        │   │
     ┌──┴───┘
     │   │
  ┌──┴───┘
  │   │
──┴───┘

3개의 연속 양봉
각 종가 > 이전 종가
긴 몸통, 짧은 꼬리
```

### 세 까마귀 (Three Black Crows)

```
약세 지속 신호

──┬███┐
  │███│
  └┬██┴──┐
   │█████│
   └┬████┴──┐
    │███████│
    └███████┘

3개의 연속 음봉
각 종가 < 이전 종가
긴 몸통, 짧은 꼬리
```

## 패턴 신뢰도 높이기

### 확인 방법

```
1. 추세 맥락
   └── 상승 반전 패턴은 하락추세에서
   └── 하락 반전 패턴은 상승추세에서

2. 지지/저항
   └── 주요 지지선에서 상승 반전 패턴
   └── 주요 저항선에서 하락 반전 패턴

3. 거래량
   └── 반전 패턴 + 거래량 증가 = 신뢰도 ↑

4. 확인 캔들
   └── 다음 캔들이 방향 확인해야 진입
```

### 패턴별 신뢰도

| 패턴 | 신뢰도 | 확인 필요 |
|------|--------|----------|
| 장악형 | ★★★★★ | 낮음 |
| 샛별/저녁별 | ★★★★★ | 낮음 |
| 망치/슈팅스타 | ★★★★ | 중간 |
| 하라미 | ★★★ | 높음 |
| 도지 | ★★ | 매우 높음 |

## 실전 적용 팁

### 체크리스트

```
□ 현재 추세 확인
□ 지지/저항 레벨 확인
□ 패턴 출현 위치 확인
□ 거래량 확인
□ 확인 캔들 대기
□ 손절선 설정
□ 목표가 설정
```

### 흔한 실수

1. **맥락 무시** - 추세 반대 패턴 무시
2. **조기 진입** - 확인 전 진입
3. **손절 미설정** - 패턴 실패 대비 없음
4. **과신** - 하나의 패턴에만 의존
