# 기술적 분석 개요

> "가격은 모든 것을 반영한다."

## 기술적 분석이란?

**기술적 분석 (Technical Analysis)** 은 과거 가격과 거래량 데이터를 분석하여 미래 가격 움직임을 예측하는 방법입니다.

## 기본 가정

### 3가지 핵심 전제

```
1. 시장 가격은 모든 정보를 반영한다
   └── 펀더멘털, 심리, 뉴스 모두 가격에 포함

2. 가격은 추세를 따라 움직인다
   └── 추세는 반전 신호가 나타날 때까지 지속

3. 역사는 반복된다
   └── 인간 심리는 변하지 않음 → 패턴 반복
```

### 펀더멘털 분석 vs 기술적 분석

| 구분 | 펀더멘털 분석 | 기술적 분석 |
|------|-------------|------------|
| 대상 | 재무제표, 산업, 경제 | 가격, 거래량, 차트 |
| 질문 | "무엇을" 살 것인가? | "언제" 살/팔 것인가? |
| 시간 | 장기 | 단기-중기 |
| 장점 | 내재가치 파악 | 타이밍 포착 |
| 단점 | 타이밍 어려움 | 가치 판단 부재 |

## 기술적 분석의 구성 요소

### 1. 차트 유형

```
┌─────────────────────────────────────────────┐
│              차트 유형                       │
├─────────────┬─────────────┬─────────────────┤
│   라인차트   │   바차트    │   캔들차트      │
│             │             │                 │
│    ╱╲      │   │─┤      │   ┃  █  ┃       │
│   ╱  ╲     │   ├─│      │   ┃  █  ┃       │
│  ╱    ╲    │   │─┤      │   ┃  █  ┃       │
│             │             │                 │
│  종가만    │ 시/고/저/종 │ 시/고/저/종     │
│  표시      │  표시       │  + 음봉/양봉    │
└─────────────┴─────────────┴─────────────────┘
```

### 2. 시간 프레임

| 기간 | 용도 | 트레이딩 스타일 |
|------|------|----------------|
| 1분, 5분 | 스캘핑 | 초단타 |
| 15분, 1시간 | 데이트레이딩 | 당일 매매 |
| 4시간, 일봉 | 스윙 | 수일-수주 |
| 주봉, 월봉 | 포지션 | 수개월-수년 |

### 3. 분석 도구

```
기술적 분석 도구
├── 추세 분석
│   ├── 추세선
│   ├── 이동평균선
│   └── 채널
│
├── 패턴 분석
│   ├── 캔들 패턴
│   ├── 차트 패턴
│   └── 가격 패턴
│
├── 지표 분석
│   ├── 추세 지표 (MA, MACD)
│   ├── 모멘텀 지표 (RSI, Stochastic)
│   └── 거래량 지표 (OBV, Volume)
│
└── 지지/저항
    ├── 수평 지지/저항
    ├── 피보나치
    └── 피벗 포인트
```

## 추세의 이해

### 추세의 3가지 방향

```
상승추세 (Uptrend)         횡보 (Sideways)         하락추세 (Downtrend)
      ╱╲                   ─────────────              ╲
     ╱  ╲╱╲                                            ╲╱╲
    ╱     ╲╱╲                                            ╲╱╲
   ╱        ╲                                               ╲

Higher Highs           Range-bound              Lower Highs
Higher Lows                                     Lower Lows
```

### 추세의 3단계

다우 이론(Dow Theory)에 따른 추세 단계:

**상승장:**
```
1. 축적기 (Accumulation)
   - 스마트머니 매집
   - 대중 비관적
   - 거래량 낮음

2. 참여기 (Public Participation)
   - 추세 확인
   - 대중 참여 시작
   - 거래량 증가

3. 과열기 (Excess)
   - 대중 탐욕
   - 뉴스 매우 긍정적
   - 스마트머니 매도 시작
```

## 지지와 저항

### 개념

```
저항 (Resistance) ─────────────────
                   ↓ 매도 압력
        가격
                   ↑ 매수 압력
지지 (Support)   ─────────────────

지지: 하락을 멈추게 하는 가격대
저항: 상승을 멈추게 하는 가격대
```

### 역할 전환

```
돌파 전:          돌파 후:

───저항───       ───────────
    ↓                 ↑
   가격          ───지지───
    ↑
───지지───

저항 돌파 → 지지로 전환
지지 붕괴 → 저항으로 전환
```

## 거래량 분석

### 거래량의 중요성

> "가격은 거짓말할 수 있지만, 거래량은 거짓말하지 않는다."

| 가격 | 거래량 | 의미 |
|------|--------|------|
| 상승 | 증가 | 강세 확인 |
| 상승 | 감소 | 상승 약화 |
| 하락 | 증가 | 약세 확인 |
| 하락 | 감소 | 하락 약화 |

### 거래량 패턴

```
클라이맥스 (Climax):
극단적 거래량 = 추세 전환 가능성

│ ││
│ ││
│ ││      거래량 급증
│ ││ │
│││││││
└──────────
  ↑
반전 시점
```

## 기술적 분석의 한계

### 주의점

1. **자기실현적 예언** - 많은 사람이 같은 패턴을 보면 실현됨
2. **후행성** - 지표는 과거 데이터 기반
3. **주관성** - 같은 차트를 다르게 해석 가능
4. **시장 조작** - 패턴이 인위적으로 만들어질 수 있음

### 효과적인 사용법

```
✓ 펀더멘털 분석과 병행
✓ 여러 시간대 확인 (멀티 타임프레임)
✓ 다양한 지표로 교차 확인
✓ 손절선 반드시 설정
✓ 리스크 관리 철저
```

## 학습 로드맵

```
Level 1: 기초
├── 캔들차트 읽기
├── 추세선 그리기
└── 지지/저항 찾기

Level 2: 중급
├── 이동평균선
├── RSI, MACD
└── 차트 패턴

Level 3: 고급
├── 피보나치
├── 엘리엇 파동
└── 복합 지표 전략
```
