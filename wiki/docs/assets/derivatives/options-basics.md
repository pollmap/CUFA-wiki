# 옵션 기초

## 옵션이란?

**옵션 (Option)** 은 특정 자산을 미래 특정 시점에 미리 정한 가격으로 사거나 팔 수 있는 **권리**입니다.

### 콜옵션 vs 풋옵션

```
콜옵션 (Call Option):
"살 수 있는 권리"
- 기초자산 가격 상승 시 이익
- 매수자: 상승 기대
- 매도자: 횡보/하락 기대

풋옵션 (Put Option):
"팔 수 있는 권리"
- 기초자산 가격 하락 시 이익
- 매수자: 하락 기대
- 매도자: 횡보/상승 기대
```

### 옵션의 구성 요소

| 요소 | 설명 |
|------|------|
| 기초자산 (Underlying) | 옵션 행사 대상 (주식, 지수 등) |
| 행사가격 (Strike/Exercise Price) | 거래 가격 |
| 만기일 (Expiration Date) | 권리 행사 기한 |
| 프리미엄 (Premium) | 옵션 가격 |
| 옵션 유형 | 콜 또는 풋 |

### 옵션 스타일

```
미국형 (American):
- 만기 전 언제든 행사 가능
- 조기행사 가치 포함
- 프리미엄 높음

유럽형 (European):
- 만기일에만 행사 가능
- 분석이 더 단순
- 프리미엄 낮음

한국 시장:
- KOSPI200 옵션: 유럽형
- 개별주식 옵션: 유럽형
```

## 옵션의 손익 구조

### 콜옵션 매수 (Long Call)

```
손익
  │
  │         ╱
  │        ╱
  │       ╱
──┼──────X─────────── 기초자산 가격
  │     ╱│
  │____╱ │ ← 손익분기점 = 행사가 + 프리미엄
 -프리미엄│
  │      행사가

최대 손실: 프리미엄
최대 이익: 무제한
```

### 풋옵션 매수 (Long Put)

```
손익
  │
  │╲
  │ ╲
  │  ╲
──┼───X────────────── 기초자산 가격
  │   │╲
  │   │ ╲__________
  │   │ -프리미엄
  │   행사가

최대 손실: 프리미엄
최대 이익: 행사가 - 프리미엄 (기초자산 0일 때)
```

### 콜옵션 매도 (Short Call)

```
손익
  │
  │____
  │    ╲╲
  │     ╲╲
──┼──────X─────────── 기초자산 가격
  │      ╲│
  │       ╲
  │        ╲

최대 이익: 프리미엄
최대 손실: 무제한
```

### 풋옵션 매도 (Short Put)

```
손익
  │      __________
  │     ╱
  │    ╱
──┼───X────────────── 기초자산 가격
  │  ╱│
  │ ╱ │
  │╱  │

최대 이익: 프리미엄
최대 손실: 행사가 - 프리미엄
```

## 옵션의 가치

### 내재가치 (Intrinsic Value)

```
즉시 행사 시 얻을 수 있는 가치

콜옵션: max(S - K, 0)
풋옵션: max(K - S, 0)

S = 기초자산 가격
K = 행사가격
```

### 시간가치 (Time Value)

```
프리미엄 = 내재가치 + 시간가치

시간가치 = 프리미엄 - 내재가치

시간가치 특성:
- 만기가 멀수록 큼
- ATM에서 최대
- 만기 근접 시 급감 (시간가치 소멸)
```

### 내가격/등가격/외가격 (ITM/ATM/OTM)

```
콜옵션:
┌──────────────────────────────────────┐
│  ITM (내가격)  │  ATM  │  OTM (외가격) │
│    S > K      │ S ≈ K │    S < K      │
│   내재가치 有  │       │  내재가치 無   │
└──────────────────────────────────────┘

풋옵션:
┌──────────────────────────────────────┐
│  OTM (외가격)  │  ATM  │  ITM (내가격) │
│    S > K      │ S ≈ K │    S < K      │
│  내재가치 無   │       │   내재가치 有  │
└──────────────────────────────────────┘
```

## 옵션 프리미엄 결정 요인

### 6가지 결정 요인

| 요인 | 콜옵션 | 풋옵션 |
|------|--------|--------|
| 기초자산 가격 ↑ | ↑ | ↓ |
| 행사가격 ↑ | ↓ | ↑ |
| 잔존만기 ↑ | ↑ | ↑ |
| 변동성 ↑ | ↑ | ↑ |
| 금리 ↑ | ↑ | ↓ |
| 배당 ↑ | ↓ | ↑ |

### 변동성의 중요성

```
변동성 ↑ → 프리미엄 ↑

이유:
- 옵션 매수자는 손실이 제한됨 (프리미엄)
- 이익은 무제한 (콜) 또는 행사가까지 (풋)
- 변동성 클수록 큰 이익 기회 ↑
- 따라서 프리미엄 가치 ↑
```

## 옵션 거래 전략

### 기본 전략

```
시장 전망별 전략:

강세 (Bullish):
├── 콜 매수
├── 풋 매도
└── 콜 스프레드

약세 (Bearish):
├── 풋 매수
├── 콜 매도
└── 풋 스프레드

횡보 (Neutral):
├── 스트래들 매도
├── 스트랭글 매도
└── 아이언 콘도르

변동성 기대:
├── 스트래들 매수
└── 스트랭글 매수
```

### 스프레드 전략

```
불 콜 스프레드 (Bull Call Spread):
- 낮은 행사가 콜 매수
- 높은 행사가 콜 매도
- 비용 절감, 이익 제한

손익:
        ___
       ╱
      ╱
─────X────X──────
    K1   K2
```

```
베어 풋 스프레드 (Bear Put Spread):
- 높은 행사가 풋 매수
- 낮은 행사가 풋 매도
- 비용 절감, 이익 제한

손익:
___
   ╲
    ╲
────X────X──────
   K1   K2
```

### 스트래들/스트랭글

```
스트래들 매수 (Long Straddle):
- 동일 행사가 콜+풋 매수
- 큰 변동성 기대

손익:
    ╲  ╱
     ╲╱
─────X──────
     K

스트랭글 매수 (Long Strangle):
- 다른 행사가 콜+풋 매수
- 더 저렴, 더 큰 변동 필요

손익:
   ╲    ╱
    ╲__╱
────X──X────
   K1  K2
```

## 옵션 투자 체크리스트

### 진입 전 확인

```
□ 기초자산 분석 완료
□ 시장 방향 전망 수립
□ 변동성 수준 확인 (IV vs HV)
□ 만기 선택 (시간가치 고려)
□ 행사가 선택 (ITM/ATM/OTM)
□ 최대 손실 계산
□ 손익분기점 확인
□ 포지션 크기 결정
```

### 리스크 관리

```
□ 매수: 프리미엄 전액 손실 가능
□ 매도: 이론적 무한 손실 가능
□ 시간가치 소멸 속도 인지
□ 변동성 변화 영향 이해
□ 조기 청산 계획 수립
```
