# 옵션 그릭스 (Greeks)

## 그릭스란?

**그릭스 (Greeks)** 는 옵션 가격이 각 요인의 변화에 얼마나 민감하게 반응하는지를 나타내는 지표입니다.

```
옵션 가격 = f(S, K, T, σ, r, D)

S: 기초자산 가격   → 델타(Δ), 감마(Γ)
T: 잔존 만기       → 세타(Θ)
σ: 변동성         → 베가(ν)
r: 금리           → 로(ρ)
```

## 델타 (Delta, Δ)

### 정의

```
델타 = 기초자산 가격 변화에 대한 옵션 가격 변화

Δ = ∂V / ∂S

콜옵션: 0 ≤ Δ ≤ 1
풋옵션: -1 ≤ Δ ≤ 0
```

### 델타 특성

```
        델타
          │
        1 │     ___콜옵션___
          │   ╱
          │  ╱
        0.5├─X─────────────
          │  ╲
          │   ╲
       -1 │     ___풋옵션___
          └────────────────── 기초자산 가격
               ATM

ITM → |Δ| ≈ 1
ATM → |Δ| ≈ 0.5
OTM → |Δ| ≈ 0
```

### 델타의 활용

```
1. 방향성 노출 (Directional Exposure)
   Δ = 0.5 콜 보유 = 기초자산 50% 보유와 유사

2. 헷지 비율 (Hedge Ratio)
   주식 100주 보유 + Δ=-0.5 풋 200계약 = 델타 중립

3. 행사 확률 (Probability Proxy)
   Δ ≈ ITM으로 만기 도달 확률 (대략적)
```

### 델타 중립 (Delta Neutral)

```
포트폴리오 델타 = 0

예시:
- 주식 100주 (Δ=100)
- 콜 매도 2계약 (Δ=-0.5×100×2=-100)
- 총 델타 = 100 - 100 = 0

→ 기초자산 방향에 중립, 변동성/시간에 노출
```

## 감마 (Gamma, Γ)

### 정의

```
감마 = 기초자산 가격 변화에 대한 델타 변화

Γ = ∂Δ / ∂S = ∂²V / ∂S²

항상 양수 (콜/풋 모두)
ATM에서 최대
```

### 감마 특성

```
        감마
          │
          │      ╱╲
          │     ╱  ╲
          │    ╱    ╲
          │   ╱      ╲
          │__╱        ╲__
          └────────────────── 기초자산 가격
                ATM

ATM에서 최대
만기 근접 시 ATM 감마 급등
```

### 감마의 의미

```
높은 감마:
- 델타가 빠르게 변함
- 잦은 리밸런싱 필요
- 기회이자 위험

감마 포지션:
- Long Gamma: 옵션 매수자
  → 기초자산 큰 움직임에 유리

- Short Gamma: 옵션 매도자
  → 기초자산 작은 움직임에 유리
  → 급변동 시 큰 손실 위험
```

## 세타 (Theta, Θ)

### 정의

```
세타 = 시간 경과에 따른 옵션 가격 변화

Θ = ∂V / ∂T

일반적으로 음수 (시간가치 소멸)
옵션 매수자에게 불리
옵션 매도자에게 유리
```

### 세타 특성

```
        │세타│
          │
          │           ╱
          │          ╱
          │         ╱
          │     ___╱
          │____╱
          └────────────────── 잔존만기
           0일  30일  60일

만기 근접 시 세타 급증
ATM 옵션에서 세타 최대
```

### 세타 소멸 (Time Decay)

```
일일 시간가치 소멸:

Θ = -5 인 옵션
→ 매일 약 5원씩 가치 감소

시간가치 소멸 패턴:
- 만기 60일+: 완만한 소멸
- 만기 30일: 가속
- 만기 7일 이내: 급격한 소멸
```

## 베가 (Vega, ν)

### 정의

```
베가 = 내재변동성 변화에 대한 옵션 가격 변화

ν = ∂V / ∂σ

항상 양수 (콜/풋 모두)
변동성 ↑ → 옵션 가치 ↑
```

### 베가 특성

```
        베가
          │
          │      ___
          │   __╱   ╲__
          │__╱         ╲__
          └────────────────── 기초자산 가격
                ATM

ATM에서 최대
만기가 길수록 베가 큼
```

### 베가의 활용

```
변동성 매매 (Volatility Trading):

Long Vega (변동성 상승 기대):
- 스트래들/스트랭글 매수
- 옵션 매수 + 델타 헷지

Short Vega (변동성 하락 기대):
- 스트래들/스트랭글 매도
- 옵션 매도 + 델타 헷지

내재변동성 vs 역사적 변동성:
IV > HV → 옵션 고평가 가능
IV < HV → 옵션 저평가 가능
```

## 로 (Rho, ρ)

### 정의

```
로 = 금리 변화에 대한 옵션 가격 변화

ρ = ∂V / ∂r

콜옵션: ρ > 0 (금리 ↑ → 콜 가치 ↑)
풋옵션: ρ < 0 (금리 ↑ → 풋 가치 ↓)
```

### 로 특성

```
일반적으로 영향 작음 (단기 옵션)
장기 옵션에서 중요성 증가
금리 변동기에 주목
```

## 그릭스 요약표

| 그릭 | 측정 대상 | 콜 | 풋 | ATM 특성 |
|------|----------|-----|-----|----------|
| 델타 (Δ) | 기초자산 가격 | 0~1 | -1~0 | ≈0.5 |
| 감마 (Γ) | 델타 변화 | + | + | 최대 |
| 세타 (Θ) | 시간 경과 | - | - | 최대 절대값 |
| 베가 (ν) | 변동성 | + | + | 최대 |
| 로 (ρ) | 금리 | + | - | 중간 |

## 그릭스 포지션 관리

### 포트폴리오 그릭스

```
총 델타 = Σ(각 포지션 델타 × 수량)
총 감마 = Σ(각 포지션 감마 × 수량)
...

예시:
콜 매수 10계약 (Δ=0.5, Γ=0.03, Θ=-2, ν=10)
풋 매수 5계약 (Δ=-0.4, Γ=0.02, Θ=-1.5, ν=8)

총 Δ = 0.5×1000 - 0.4×500 = 300
총 Γ = 0.03×1000 + 0.02×500 = 40
총 Θ = -2×10 - 1.5×5 = -27.5
총 ν = 10×10 + 8×5 = 140
```

### 그릭스 중립화

```
델타 중립: 기초자산으로 헷지
- Δ=300 → 주식 300주 매도

감마 중립: 다른 옵션으로 헷지
- 긴 감마 보유 → 옵션 매도로 상쇄

베가 중립: 다른 만기 옵션으로 헷지
- 긴 베가 보유 → 옵션 매도로 상쇄
```

## 실전 활용

### 전략별 그릭스 프로파일

| 전략 | Δ | Γ | Θ | ν |
|------|---|---|---|---|
| 콜 매수 | + | + | - | + |
| 풋 매수 | - | + | - | + |
| 콜 매도 | - | - | + | - |
| 스트래들 매수 | 0 | + | - | + |
| 불 콜 스프레드 | + | 작음 | 작음 | 작음 |

### 체크리스트

```
포지션 진입 전:
□ 각 그릭스 계산
□ 델타 노출 확인
□ 감마 위험 평가
□ 세타 소멸 영향 계산
□ 베가 노출 확인

포지션 보유 중:
□ 그릭스 변화 모니터링
□ 리밸런싱 필요 여부
□ 목표 그릭스 유지
```
