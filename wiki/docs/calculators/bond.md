# 채권 가치평가 계산기

채권 가격, 만기수익률(YTM), 듀레이션을 계산하는 도구입니다.

## 채권 가격 공식

<div className="formula-block">

**P = Σ C/(1+r)^t + F/(1+r)^n**

- P: 채권 가격
- C: 쿠폰 (이자)
- F: 액면가
- r: 할인율 (시장금리)
- n: 만기

</div>

## 주요 지표

### 만기수익률 (YTM)

채권을 만기까지 보유할 때 얻는 연환산 수익률입니다.

```
YTM > 표면금리 → 할인채 (가격 < 액면가)
YTM = 표면금리 → 액면채 (가격 = 액면가)
YTM < 표면금리 → 할증채 (가격 > 액면가)
```

### 듀레이션 (Duration)

금리 변동에 대한 채권 가격의 민감도를 나타냅니다.

| 유형 | 설명 |
|------|------|
| 맥컬리 듀레이션 | 가중평균 현금흐름 회수 기간 (년) |
| 수정 듀레이션 | 금리 1% 변동 시 가격 변동률 |
| 볼록성 | 2차 민감도 (비선형 조정) |

### 가격 변동 추정

```
ΔP/P ≈ -D × Δr + 0.5 × C × (Δr)²

D: 수정 듀레이션
C: 볼록성
Δr: 금리 변동 (%)
```

## 계산 예시

| 입력 | 값 |
|------|-----|
| 액면가 | 10,000원 |
| 표면금리 | 3% |
| 만기 | 5년 |
| 시장금리 | 4% |
| 이자지급 | 반기 |

| 결과 | 값 |
|------|-----|
| 채권 가격 | 9,554.44원 |
| 경상수익률 | 3.14% |
| 맥컬리 듀레이션 | 4.63년 |
| 수정 듀레이션 | 4.54 |
