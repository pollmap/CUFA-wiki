# 하워드 막스 (Howard Marks)

> "투자에서 가장 중요한 것은 무엇이 좋은지 아는 게 아니라, 무엇이 가격에 반영되어 있는지 아는 것이다."

## 인물 소개

**하워드 막스 (1946~)** 는 오크트리 캐피탈의 공동 창립자로, **부실채권 투자**의 대가이며 투자 메모로 유명합니다. 워렌 버핏도 그의 메모가 오면 가장 먼저 읽는다고 합니다.

| 항목 | 내용 |
|------|------|
| 운용사 | 오크트리 캐피탈 매니지먼트 |
| 설립 | 1995년 |
| 운용 자산 | $170B+ |
| 전문 분야 | 부실채권, 하이일드 |
| 대표 저서 | 《투자에 대한 생각》, 《시장 사이클의 법칙》 |

## 핵심 투자 철학

### 1. 2차적 사고 (Second-Level Thinking)

1차적 사고를 넘어 **한 단계 더 깊이** 생각해야 합니다.

| 1차적 사고 | 2차적 사고 |
|-----------|-----------|
| "좋은 회사니까 매수" | "좋은 회사지만 기대가 이미 가격에 반영됨" |
| "전망이 어두우니 매도" | "모두가 비관적이라 가격이 충분히 낮음" |
| "금리가 오르니 채권 매도" | "금리 상승이 예상보다 적으면?" |
| "경기 침체가 오니 현금 보유" | "이미 주가가 침체를 반영했다면?" |

**2차적 사고 질문:**
```
1. 컨센서스는 무엇인가?
2. 내 의견은 컨센서스와 어떻게 다른가?
3. 컨센서스가 틀릴 확률은?
4. 내가 틀릴 확률은?
5. 컨센서스가 틀렸을 때 얼마나 벌 수 있나?
6. 내가 틀렸을 때 얼마나 잃나?
```

### 2. 시장 사이클

> "사이클을 예측할 수는 없지만, 어디에 있는지는 알 수 있다."

**시장 심리 사이클:**
```
                      탐욕
                        │
    낙관 ──────────────┼──────────────→
         ↗            │            ↘
       /              │              \
      /    환희      │     걱정      \
     /                │                \
────┼────────────────┼────────────────┼────
    /                 │                 \
   /      희망       │      공포       \
  ↗                  │                  ↘
비관 ←───────────────┼───────────────── 절망
                      │
                    공황
```

### 3. 위험의 정의

막스는 전통적 위험 정의에 이의를 제기합니다.

| 전통적 정의 | 막스의 정의 |
|------------|------------|
| 변동성 (σ) | **영구적 자본 손실 확률** |
| 측정 가능 | 측정 어려움 |
| 과거 기반 | 미래 지향 |
| 베타 | 펀더멘털 약화, 과대 평가 |

**위험의 역설:**
```
인식된 위험 ↑ → 실제 위험 ↓ (가격 하락으로)
인식된 위험 ↓ → 실제 위험 ↑ (가격 상승으로)

"위험은 가장 안전해 보일 때 가장 크다"
```

## 시장 사이클의 법칙

### 사이클의 3가지 특성

1. **피할 수 없다** - 인간 심리가 만듦
2. **극단으로 간다** - 평균에 머물지 않음
3. **회귀한다** - 결국 평균으로 돌아옴

### 사이클 위치 파악

**낙관적 시기 (위험 신호):**
```
□ "이번엔 다르다" 논리 확산
□ 신규 투자자 대거 유입
□ 레버리지 증가
□ 기업공개(IPO) 급증
□ 언론의 긍정적 보도 증가
□ 가치평가 정당화 논리 등장
□ "기회를 놓치면 안 된다" 심리
```

**비관적 시기 (기회 신호):**
```
□ "회복 불가능" 논리 확산
□ 투매성 매도
□ 펀드 환매 급증
□ 우량 자산도 폭락
□ 언론의 부정적 보도 일색
□ "절대 회복 못한다" 심리
□ 아무도 매수하려 하지 않음
```

### 막스의 포트폴리오 조절

| 사이클 위치 | 포지션 |
|------------|--------|
| 사이클 저점 | 공격적 (100% 투자, 레버리지 가능) |
| 평균 | 중립적 (정상 배분) |
| 사이클 고점 | 방어적 (현금 증가, 품질 상향) |

## 투자에서 가장 중요한 것

막스의 책 《투자에 대한 생각》의 핵심 20가지:

### 사고 (Thinking)

1. **2차적 사고** - 한 단계 더 깊이
2. **시장 효율성 이해** - 완전하지도, 무의미하지도 않음
3. **가치 이해** - 내재가치 추정 능력
4. **가격과 가치의 관계** - 좋은 매수 ≠ 좋은 기업
5. **위험 이해** - 변동성이 아닌 손실 가능성

### 실행 (Action)

6. **위험 인식** - 위험은 숨어 있다
7. **위험 통제** - 손실 회피가 수익보다 중요
8. **사이클 주의** - 역사는 반복된다
9. **진자 인식** - 극단에서 기회
10. **심리적 영향 대응** - 감정 통제

### 역발상 (Contrarian)

11. **역발상 투자** - 컨센서스에 맞서기
12. **저가 매수** - 떨어지는 칼 잡기
13. **인내** - 시간이 필요
14. **무지의 인식** - 모르는 것을 알기
15. **예측의 한계** - 미래는 불확실

### 방어 (Defense)

16. **합리적 기대** - 현실적 수익률
17. **방어적 투자** - 공격보다 방어
18. **함정 피하기** - 손실 회피
19. **부가가치 창출** - 알파 추구
20. **모든 것의 통합** - 종합적 접근

## 체크리스트: 막스 스타일 투자

### 매수 전 확인

```
□ 컨센서스와 다른 관점이 있는가?
□ 내 관점이 틀릴 경우 손실은 얼마인가?
□ 현재 시장 사이클 위치는?
□ 가격이 비관론을 충분히 반영했는가?
□ 영구적 자본 손실 위험은 낮은가?
□ 인내심을 가질 수 있는가?
```

### 매도 신호

```
□ 낙관론이 가격에 완전히 반영됨
□ "이번엔 다르다" 논리 등장
□ 더 나은 위험/보상 기회 발견
□ 투자 논리(thesis)가 무효화됨
```

## 부실채권 투자

막스의 전문 분야인 부실채권 투자:

### 기회 발생 시점

```
1. 과도한 신용 확장 → 버블
2. 버블 붕괴 → 패닉 매도
3. 우량 채권도 동반 하락
4. 회복 가능한 채권을 저가 매수
5. 시장 정상화 시 큰 수익
```

### 부실채권 분석

| 확인 항목 | 내용 |
|----------|------|
| 기업 가치 | 청산가치 vs 계속기업가치 |
| 자본구조 | 선순위 vs 후순위 |
| 회생 가능성 | 구조조정 여력 |
| 회수율 | 원금 대비 회수 예상 |
| 타이밍 | 정상화 예상 시점 |

## 불확실성에 대한 대응

### 알 수 있는 것 vs 알 수 없는 것

```
알 수 있는 것:
- 현재 가치평가 수준
- 현재 시장 심리
- 역사적 패턴
- 기업의 현재 펀더멘털

알 수 없는 것:
- 정확한 미래
- 사이클의 정확한 전환점
- 외부 충격 (블랙스완)
- 시장 타이밍
```

### 불확실성 대응 원칙

1. **겸손** - 모른다는 것을 인정
2. **분산** - 한 곳에 집중 금지
3. **안전마진** - 틀려도 괜찮은 가격
4. **인내** - 시간의 편에 서기
