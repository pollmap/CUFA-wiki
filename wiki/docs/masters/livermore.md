# 제시 리버모어 (Jesse Livermore)

> "시장은 절대 틀리지 않는다. 의견은 종종 틀린다."

## 인물 소개

**제시 리버모어 (1877-1940)** 는 "월가의 곰"으로 불린 전설적인 투기자로, 1929년 대공황에서 공매도로 **1억 달러** (현재 가치 약 15억 달러)를 벌었습니다.

| 항목 | 내용 |
|------|------|
| 활동 기간 | 1890년대 - 1940년 |
| 별명 | 월가의 곰 (The Great Bear) |
| 대표 투자 | 1929년 대공황 공매도 |
| 최대 자산 | $100M (1929년) |
| 대표 저서 | 《어느 주식투자자의 회상》 |

## 핵심 투자 철학

### 1. 추세추종 (Trend Following)

> "큰돈은 기다림에서 나온다. 매매에서가 아니다."

**핵심 원칙:**
```
1. 추세를 따르라 - 역행하지 마라
2. 상승장에서는 롱, 하락장에서는 숏
3. 추세가 바뀔 때까지 포지션 유지
4. 가격이 방향을 알려준다
```

**추세 확인 방법:**
```
상승추세:
- 고점이 점점 높아짐 (Higher Highs)
- 저점도 점점 높아짐 (Higher Lows)

하락추세:
- 고점이 점점 낮아짐 (Lower Highs)
- 저점도 점점 낮아짐 (Lower Lows)
```

### 2. 피라미딩 (Pyramiding)

수익이 나는 포지션에 **추가 매수**하는 전략입니다.

```
1차 매수: 100주 @ $50 (테스트)
    │
    ▼ 주가 상승 확인
    │
2차 매수: 80주 @ $55 (확인)
    │
    ▼ 추세 지속 확인
    │
3차 매수: 60주 @ $60 (추가)
    │
    ▼
총 포지션: 240주, 평균 $54.6
```

**피라미딩 규칙:**
- 수익 나는 포지션에만 추가
- 추가 매수량은 점점 줄임
- 손실 포지션에는 절대 추가 금지

### 3. 손절의 중요성

> "손실을 작게, 수익을 크게"

**리버모어의 손절 규칙:**
```
1. 10% 손실 시 무조건 청산
2. 변명하지 마라
3. 손실은 투자의 비용
4. 다음 기회는 반드시 온다
```

**손실 관리:**
```
원금 $100,000
├── 10% 손실 → $90,000 (11% 수익으로 회복)
├── 25% 손실 → $75,000 (33% 수익 필요)
├── 50% 손실 → $50,000 (100% 수익 필요)
└── 75% 손실 → $25,000 (300% 수익 필요)

→ 작은 손실일 때 끊어야 회복 가능
```

## 리버모어의 매매 규칙

### 진입 규칙

| # | 규칙 | 설명 |
|---|------|------|
| 1 | **타이밍** | 시장 전체 방향과 일치할 때만 진입 |
| 2 | **확인** | 가격이 방향을 확인할 때까지 대기 |
| 3 | **분할 매수** | 한 번에 전량 매수 금지 |
| 4 | **피벗 포인트** | 핵심 가격대 돌파 시 진입 |

### 청산 규칙

| # | 규칙 | 설명 |
|---|------|------|
| 1 | **손절** | 10% 이상 손실 시 즉시 청산 |
| 2 | **추세 전환** | 추세 반전 신호 시 청산 |
| 3 | **목표 달성** | 미리 설정한 목표가 도달 시 |
| 4 | **불확실** | 확신이 없으면 청산 |

### 피벗 포인트 (Pivot Point)

리버모어가 중시한 핵심 가격대:

```
         저항선 ───────────────
              │
      피벗 포인트 (돌파 시 매수)
              │
         지지선 ───────────────

매수 신호: 피벗 포인트 상향 돌파 + 거래량 증가
매도 신호: 피벗 포인트 하향 붕괴 + 거래량 증가
```

## 시장 심리 이해

### 군중 심리의 단계

```
1. 축적기 (Accumulation)
   - 스마트머니 매집
   - 뉴스: 부정적
   - 대중: 무관심

2. 상승기 (Markup)
   - 추세 확인
   - 뉴스: 중립 → 긍정적
   - 대중: 관심 시작

3. 분산기 (Distribution)
   - 스마트머니 매도
   - 뉴스: 매우 긍정적
   - 대중: 탐욕, FOMO

4. 하락기 (Markdown)
   - 패닉 매도
   - 뉴스: 부정적
   - 대중: 공포
```

### 리버모어의 심리 원칙

> "인간의 본성은 시장에서 성공에 가장 큰 적이다."

**극복해야 할 감정:**
1. **희망** - 손실 포지션 유지의 원인
2. **공포** - 이익 실현을 너무 빨리
3. **탐욕** - 과도한 레버리지
4. **후회** - 판단력 흐림

## 공매도 (Short Selling)

리버모어가 "월가의 곰"이 된 이유:

### 공매도 타이밍

```
최적의 공매도 시점:
□ 시장이 과열 상태
□ 가격이 펀더멘털에서 크게 괴리
□ 거래량 감소하며 신고가
□ 낙관론 만연
□ 레버리지 증가
□ 신규 투자자 대거 유입
```

### 1929년 공매도

```
1929년 리버모어의 행동:
1. 여름: 시장 과열 감지
2. 9월: 경고 신호 포착
3. 10월: 대규모 공매도 포지션
4. 10월 29일 (블랙 화요일): 시장 붕괴
5. 결과: $100M 이익

핵심: 군중이 탐욕에 빠질 때 두려워하라
```

## 리버모어의 실패와 교훈

리버모어는 4번 파산하고 재기했습니다:

### 실패 원인

| # | 원인 | 교훈 |
|---|------|------|
| 1 | **과도한 레버리지** | 레버리지는 양날의 검 |
| 2 | **규칙 위반** | 자신의 규칙을 지켜라 |
| 3 | **확신 과잉** | 항상 겸손하라 |
| 4 | **개인 문제** | 투자와 개인 삶 분리 |

### 재기의 비결

```
1. 작게 시작
2. 규칙으로 복귀
3. 손절 철저
4. 피라미딩으로 수익 확대
5. 확신이 있을 때만 배팅
```

## 현대적 적용

### 리버모어 전략의 현대화

| 리버모어 원칙 | 현대 도구 |
|-------------|----------|
| 추세 확인 | 이동평균선, ADX |
| 피벗 포인트 | 지지/저항, 피보나치 |
| 거래량 분석 | OBV, 거래량 지표 |
| 손절 | 자동 스톱로스 |
| 피라미딩 | 분할 매수 주문 |

### 현대 트레이더를 위한 체크리스트

```
진입 전:
□ 시장 전체 추세 확인
□ 섹터 추세 확인
□ 개별 종목 추세 확인
□ 피벗 포인트 식별
□ 손절 가격 설정
□ 포지션 크기 결정

보유 중:
□ 추세 지속 여부 확인
□ 피라미딩 기회 탐색
□ 손절선 조정 (트레일링)
□ 감정 통제

청산 시:
□ 손절선 도달?
□ 추세 반전 신호?
□ 목표가 도달?
□ 분할 청산 실행
```

## 리버모어의 하루

```
장 시작 전:
- 전일 가격 행동 분석
- 뉴스 및 시장 상황 점검
- 오늘의 계획 수립

장중:
- 가격 움직임 관찰
- 계획대로만 실행
- 감정적 매매 금지

장 마감 후:
- 오늘의 매매 복기
- 실수 분석
- 내일 계획 수립
```
