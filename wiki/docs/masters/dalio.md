# 레이 달리오 (Ray Dalio)

> "고통 + 반성 = 진보 (Pain + Reflection = Progress)"

## 인물 소개

**레이 달리오 (1949~)** 는 세계 최대 헤지펀드 브릿지워터 어소시에이츠의 창립자로, **1,500억 달러** 이상의 자산을 운용하며 체계적인 투자 원칙으로 유명합니다.

| 항목 | 내용 |
|------|------|
| 운용사 | 브릿지워터 어소시에이츠 |
| 설립 | 1975년 |
| 운용 자산 | $150B+ (2024년 기준) |
| 투자 스타일 | 글로벌 매크로, 리스크 패리티 |
| 대표 저서 | 《원칙 (Principles)》 |

## 핵심 투자 철학

### 1. 원칙 기반 의사결정 (Principles-Based)

모든 결정을 **명문화된 원칙**에 따라 체계적으로 내립니다.

**투자 원칙 예시:**
```
원칙 #1: 확신도에 비례하여 배팅하라
원칙 #2: 상관관계가 낮은 15-20개 수익원을 확보하라
원칙 #3: 무엇을 모르는지 아는 것이 더 가치있다
원칙 #4: 역사는 반복되지만 정확히 같지는 않다
```

### 2. 경제 기계 (Economic Machine)

달리오는 경제를 **기계**처럼 이해합니다.

```
거래 = 지출 = 소득 (다른 사람의)

지출 = 현금 + 신용(부채)

경제 성장 = 생산성 + 단기부채사이클 + 장기부채사이클
```

**3가지 주요 동인:**
1. **생산성 증가** - 장기 상승 추세
2. **단기 부채 사이클** - 5-10년 주기
3. **장기 부채 사이클** - 75-100년 주기

### 3. 리스크 패리티 (Risk Parity)

자산이 아닌 **위험**을 균등하게 배분합니다.

**전통적 배분 vs 리스크 패리티:**

| 전통적 60/40 | 리스크 패리티 |
|-------------|--------------|
| 주식 60% | 주식 위험 25% |
| 채권 40% | 채권 위험 25% |
| 위험의 90%가 주식 | 원자재 위험 25% |
| | 인플레 자산 위험 25% |

## 올웨더 포트폴리오 (All Weather)

모든 경제 환경에서 성과를 내는 포트폴리오입니다.

### 4가지 경제 환경

```
              인플레이션
                 ↑
    [성장↓인플레↑]  |  [성장↑인플레↑]
       스태그플레이션 |   호황/과열
    ←───────────────┼───────────────→ 성장
       디플레이션   |    골디락스
    [성장↓인플레↓]  |  [성장↑인플레↓]
                 ↓
```

### 환경별 유리한 자산

| 경제 환경 | 유리한 자산 |
|----------|-----------|
| 성장 ↑ 인플레 ↑ | 주식, 원자재, 신흥국 채권, 금 |
| 성장 ↑ 인플레 ↓ | 주식, 회사채, 국채 |
| 성장 ↓ 인플레 ↑ | 물가연동채(TIPS), 금, 원자재 |
| 성장 ↓ 인플레 ↓ | 국채, 물가연동채 |

### 올웨더 자산 배분

```
┌─────────────────────────────────────┐
│          올웨더 포트폴리오          │
├─────────────────────────────────────┤
│ 장기 국채 (20년+)        40%       │
│ 중기 국채 (7-10년)       15%       │
│ 주식 (S&P 500)           30%       │
│ 금                        7.5%      │
│ 원자재                    7.5%      │
└─────────────────────────────────────┘
```

**리밸런싱 규칙:**
- 연 1회 또는 배분 ±5% 벗어날 때
- 세금 효율적인 계좌에서 실행

## 부채 사이클 이론

### 단기 부채 사이클 (5-10년)

```
확장기 → 정점 → 침체 → 저점 → 확장기

[확장기]
- 신용 증가, 소비 증가
- 자산 가격 상승
- 낙관론 확산

[정점/침체]
- 부채 부담 증가
- 중앙은행 긴축
- 자산 가격 하락

[저점/회복]
- 디레버리징
- 중앙은행 완화
- 새로운 사이클 시작
```

### 장기 부채 사이클 (75-100년)

**4단계:**
1. **초기 부채 축적** - 생산적 투자
2. **버블 형성** - 과도한 신용 확장
3. **정점/위기** - 디레버리징 시작
4. **아름다운 디레버리징** - 균형 잡힌 조정

**역사적 사례:**
- 1930년대 대공황
- 2008년 금융위기
- 일본 잃어버린 30년

## 투자 원칙 체크리스트

### 분산 투자 원칙

```
✓ 상관관계가 낮은 15-20개 수익원 확보
✓ 특정 자산이 전체 위험의 50% 이상 차지하지 않도록
✓ 각 경제 환경에 대비한 자산 보유
✓ 레버리지는 변동성 조절에만 사용
```

### 의사결정 원칙

```
✓ 모든 결정에 근거와 확률 부여
✓ 실수를 기록하고 원칙으로 전환
✓ 신뢰도 가중 의사결정 사용
✓ 스트레스 테스트 실시
```

## 신뢰도 가중 의사결정

달리오의 핵심 원칙 중 하나입니다.

**방법론:**
```
최종 결정 = Σ(각 의견 × 신뢰도 가중치)

신뢰도 산정 기준:
1. 해당 분야 실적 (track record)
2. 논리의 깊이
3. 과거 정확도
```

**예시:**
```
주식 시장 전망
- 전문가 A (신뢰도 0.4): 상승 (+10%)
- 전문가 B (신뢰도 0.3): 보합 (+2%)
- 전문가 C (신뢰도 0.3): 하락 (-5%)

가중 전망 = 0.4×10 + 0.3×2 + 0.3×(-5) = 3.1%
```

## 개인 투자자를 위한 조언

달리오가 개인 투자자에게 권하는 원칙:

### 1. 분산하라
> "성공적인 분산은 당신이 모르는 것에 대한 최선의 보호"

### 2. 비용을 낮춰라
- 인덱스 펀드 활용
- 불필요한 거래 최소화

### 3. 시스템을 만들어라
```
1. 저축률 설정 (소득의 15%+)
2. 자산 배분 결정 (올웨더 참고)
3. 리밸런싱 규칙 설정
4. 규칙 준수 (감정 배제)
```

### 4. 겸손하라
> "가장 확신할 때가 가장 위험할 때"
